Project Description:

My project is a rendition of the classic “Snakes and Ladders” board game. The code takes two players' names and runs them through the game. If players land on a ladder, then they get to increase their position to the top of the ladder. However, if they land on a snake, then they slide down the board to the position the snake ends at. The goal of this game is to be the first player to land exactly on 100. If a player goes above 100, then they return back to their previous spot. This version of the game adds a fun twist to the ordinary “Snakes and Ladders.” Instead of using one 6-sided dice the whole game, players can choose between the normal 6-sided dice or a dice that ranges from -4 to 10 each turn. This allows players to either go forward normally or take a risk to gain a few more steps at the potential of losing some.

Instructions:

To start the game, start up your terminal. You will first be prompted to enter the name of the first player, followed by the name of the second player. Simply type in each first name when asked to continue the game. Each player will then be asked to choose between one of the two dice each turn. After inputting which dice you choose, the game will let you know where your current position is and if you hit any ladders or snakes, went above one hundred, or won the game. Have fun playing!
